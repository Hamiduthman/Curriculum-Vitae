The Basic Language of the Web

Posted by Laura Jonas on monday, June 21st 2023

All modern Website and web applications: are build using three fundamental technologies: HTML, CSS, and JavaScript. These are the language of the web

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it

What is HTML

HTML stands for Hyper Text Markup Language. It's a markup language that web developer us to structure and describe the content of a webpage (not a programming language).

And HTML consists of different elements that web developers use to describe different types of content: Paragraph, Links, images, headings, video, etc. And web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 part

  1. The opening tag
  2. The closing tag
  3. The actual element
